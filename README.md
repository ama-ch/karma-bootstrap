## Testing Frameworks

* [Mocha](http://mochajs.org/)
* [Expect.js](https://github.com/LearnBoost/expect.js/)
* [Sinon.JS](http://sinonjs.org/)

## Preparation

Before running tests, install [Karma](http://karma-runner.github.io/) as following:

```bash
$ npm install -g karma
```

## Running tests

```bash
$ # TDD
$ karma start
$ # Single run
$ npm test
```
